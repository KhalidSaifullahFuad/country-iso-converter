Country ISO Converter

Convert ISO country codes to Country names and vice versa.

How to use this tool

  • Paste or type the country names or code list items into the INPUT list text area separated by new-line.
  • Pick the options required and click on the [Convert Country List] button.
  • The results will appear in the RESULT list text area.
